FASE 1: COMPRENSIÓN DEL NEGOCIO

  • Desarrollar sistema de recomendación de películas personalizado
  • Mejorar experiencia de usuario y engagement
  • Aumentar descubrimiento de contenido relevante
  • Resolver sobrecarga de opciones (4760 películas)
  • Optimizar catálogo mediante recomendaciones inteligentes
2

FASE 2: COMPRENSIÓN DE LOS DATOS

  • Dataset: 4760 películas con metadatos completos
  • Características: género, idioma, presupuesto, popularidad
  • 1164 combinaciones de géneros diferentes
  • 37 idiomas representados
  • Ratings en escala 1.0 a 10.0
  • Datos estructurados con campos anidados (JSON)
3

FASE 3: PREPARACIÓN DE LOS DATOS

  • Extracción de campos anidados (producción, países)
  • Manejo de valores faltantes
  • Normalización de formatos de texto
  • Ingeniería de características numéricas y de texto
  • Creación de score ponderado
  • Vectorización TF-IDF para análisis de similitud
4

FASE 4: MODELADO (K-NN)

  • Algoritmo: K-Nearest Neighbors con métrica coseno
  • Enfoque: Filtrado colaborativo basado en usuarios
  • Arquitectura: Matriz de similitud entre usuarios
  • Búsqueda de vecinos: k usuarios más similares
  • Predicción: Promedio ponderado por similitud
  • Hiperparámetros: k optimizado automáticamente
5

FASE 5: EVALUACIÓN

  • Similitud Promedio: 6.6% (objetivo >30%)
  • Usuarios Aislados: 100% (objetivo <20%)
  • Cobertura: 0.52% (objetivo >50%)
  • k Óptimo: 3 (subóptimo por sparsity)
  • Problema Principal: Alta sparsity (1.14% densidad)
  • Consecuencia: Recomendaciones limitadas
6

FASE 6: DESPLIEGUE

  • Arquitectura: API REST para recomendaciones
  • Integración: Interfaz de usuario + sistema de feedback
  • Escalabilidad: Optimización para grandes volúmenes
  • Mantenimiento: Re-entrenamiento periódico
  • Monitoreo: Drift de datos y concepto
  • Actualización: Hiperparámetros según comportamiento
